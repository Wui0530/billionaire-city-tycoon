# Billionaire City Tycoon

一个 3D 城市大亨经营游戏，使用 React、Three.js 和 Vite 构建。

## 功能

- 🎮 **3D 游戏场景** - 使用 Three.js 渲染
- 🚶 **角色控制** - 使用 W/A/S/D 键移动
- 🏢 **城市建筑** - 多个不同高度和颜色的建筑
- 📱 **UI 界面** - 显示金钱、建筑数量和语言选项
- 🎥 **摄像头控制** - 使用鼠标拖拽旋转视角

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 生产构建

```bash
npm run build
```

## 控制方式

- **W** - 向前移动
- **A** - 向左移动  
- **S** - 向后移动
- **D** - 向右移动
- **鼠标拖拽** - 旋转摄像头

## 技术栈

- React
- Three.js
- @react-three/fiber
- @react-three/drei
- Vite
